# Code of Conduct

## Our Standard

Be respectful. Disagreements about code are fine — personal attacks are not.

Specifically:
- Critique code and ideas, not people.
- Be direct but constructive in reviews.
- Assume good intent by default.

## Scope

This applies to all project spaces: issues, pull requests, discussions, and any other communication in this repository.

## Enforcement

Violations can be reported to the maintainer via GitHub. Repeated or severe violations may result in a ban from the project.
